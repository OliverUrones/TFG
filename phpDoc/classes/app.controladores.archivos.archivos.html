convertir()

convertir(\app\controladores\archivos\type  $parametros = NULL)

NOTA: Esta función no se usa en ningún sitio de la aplicación ya que el proceso de convertir archivos lo he dividido en dos: 1º se suben las fotos al servidor a través del drag And Drop del formulario de forma automática con Dropzone.js 2º se manda el resto de datos del formulario (parámetros para el script NoteShrink.py) para realiar la conversión Estas dos etapas las realizan los métodos subir() y conversion().

Método que muestra el formulario de subida de archivos si viene por GET. Si viene por POST sube los archivos y manda los parámetros del script a la vez para realizar la conversión con NoteShrink.py
